    Online Judge System

    What is an Online Judge System?

    An online judge is a platform where we can submit our code for a given problem and it will return a verdict based on predefined test-cases for the problem.

    Features

    • Secure user authentication
    • Problem CRUD operations with dynamic problem-solving pages.
    • Automated code evaluation against test cases.
    • Deployment using Docker and AWS for scalability and reliability.
    • Built using the MERN stack with a focus on RESTful APIs and modular architecture.

    What is an Railway Reservation System?

    A database-driven system designed to manage railway reservations, allowing users to book and cancel tickets efficiently while automating reservation processes.

    Features

    • User-friendly booking and cancellation functionality.
    • Train schedule tracking and seat availability verification.
    • Automated updates for reservations and cancellations
    • Implementation of data structures and triggers for dynamic seat management
